1、标题的用法

```markdown
格式：#+"空格"+标题（输入几个#就代表几级标题，最小六级标题）
快捷键：Ctrl + 数字

```

# 这是一级标题

## 这是二级标题

### 这是三级标题

#### 这是四级标题

##### 这是五级标题

###### 这是六级标题

```markdown

```

[百度一下](https://www.baidu.com)

```markdown
Markdown 支持以比较简短的自动链接形式来处理网址和电子邮件信箱，只要是用<>包裹起来，Markdown 就会自动把它转成链接。一般网址的链接文字就和链接地址一样
```

5、总结

- 无序列表：- + 空格
- 有序列表：1. + 空格
- 引用：> + 空格
- 标题：ctrl + 数字
- 表格：ctrl + t
- 生成目录：[TOC] + 回车
- 选中一整行：ctrl + l
- 选中单词：ctrl + d
- 选中相同格式的文字：ctrl + e
- 跳转到文章开头：ctrl + home
- 跳转到文章结尾：ctrl + end
- 搜索：ctrl + f
- 替换：ctrl + h
- 加粗：ctrl + b
- 倾斜：ctrl + i
- 下划线：ctrl + u
- 删除线：alt + shift + 5
- 插入图片：直接拖动到指定位置即可或者ctrl + shift + i
- 插入链接：ctrl + k
